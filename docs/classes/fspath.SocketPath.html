Class SocketPath

Extension class of AbstractFsObject, which extends the native String class, representing a path to a socket.

See

AbstractFsPath

Remarks

This is a special file used for inter-process communication. It provides a communication interface between a client process and a server process. This class has no special methods or properties. It is only used to distinguish between different types of filesystem objects.
